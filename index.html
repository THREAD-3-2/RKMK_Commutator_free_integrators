Documentation of RKMK_Commutator_free_integrators¶

RKMK_Commutator_free_integrators is a MATLAB code for the implementation of Runge-Kutta-Munthe-Kaas and Commutator free Lie group integrators, which is developed at the Department of Mathematical Sciences at NTNU. It is part of the source code for the paper (Celledoni, Çokaj, Leone, Murari and Owren, 2021). The equations of motion of the mechanical system under consideration are described in Example. To solve the problem, Runge-Kutta-Munthe-Kaas and commutator free Lie group integrators are applied, as described in Algorithms. The code can be run from a main file at src/main.m.
